What is Binary

In computer science, binary is a system where numbers and values are represented by either 0 or 1, this value is called a bit. In a binary system it can represent one of two states: on or off, true or false, etc.

Binary is a base 2 numbering system

Case 1:

0001=2³+2²+2¹+2⁰

two powered by 0 = 1 so the result is 1! (Don't forget to ignore the zeros!)

Case 2:0011=2³+2²+2¹+2⁰

two powered by 0 = 1 + 2 powered by 1 = 2, so the result is 3!

Case 3:

0101=2³+2²+2¹+2⁰

two powered by 0 = 1 + 2 powered by 2 = 4, so the result is 5!

Did you know: Every time a binary ends toggled on, it means that the binary is an odd number! (When converting to decimals).
